# TIME & SPACE COMPLEXITY :
____________________________________________________________________________________________________________________
| Algorithm         | Best Case (TC) | Average Case (TC)| Worst Case (TC)| Space Complexity                        |
|-------------------|----------------|------------------|----------------|-----------------------------------------|
| 1. Bubble Sort    | O(n)           | O(n²)            | O(n²)          | O(1) (in-place)                         |
| 2. Selection Sort | O(n²)          | O(n²)            | O(n²)          | O(1) (in-place)                         |
| 3. Insertion Sort | O(n)           | O(n²)            | O(n²)          | O(1) (in-place)                         |
| 4. Merge Sort     | O(n log n)     | O(n log n)       | O(n log n)     | O(n)                                    |
| 5. Quicksort      | O(n log n)     | O(n log n)       | O(n²)          | O(log n) (stack)                        |
| 6. Counting Sort  | O(n + k)       | O(n + k)         | O(n + k)       | O(k)                                    |
| 7. Radix Sort     | O(nk)          | O(nk)            | O(nk)          | O(n + k)                                |
| 8. Bucket Sort    | O(n + k)       | O(n + k)         | O(n²)          | O(n + k)                                |
| 9. Heap Sort      | O(n log n)     | O(n log n)       | O(n log n)     | O(1) (in-place)                         |
| 10. Shell Sort    | O(n log n)     | O(n log n)       | O(n²)          | O(1) (in-place)                         |
| 11. Linear Search | O(1)           | O(n)             | O(n)           | O(1)                                    |
| 12. Binary Search | O(1)           | O(log n)         | O(log n)       | O(1) (iterative) / O(log n) (recursive) |
|------------------------------------------------------------------------------------------------------------------|
