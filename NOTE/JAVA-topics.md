# 📘 Java Learning Roadmap — Folder Structure Overview

## 📁 01-basics-and-syntax

Foundational Java concepts every developer must learn before moving to OOP.

### **Topics included**

- Java program structure (`class`, `main` method)
- Data types & variables
- Input/output using `Scanner`
- Operators
- Conditional statements (`if`, `else if`, `switch`)
- Loops (`for`, `while`, `do-while`)
- Methods (functions)
- Method overloading
- Type casting (implicit & explicit)
- Comments & coding style
- Basic arrays

---

## 📁 02-oops-concepts

Object-Oriented Programming — the backbone of Java development.

### *Topics included*

- Classes & Objects
- Constructors
- `this` & `super` keywords
- Encapsulation (getters/setters)
- Inheritance
- Method overriding
- Polymorphism
- Abstraction
- Interfaces
- Abstract classes
- Static vs Instance members
- Final keyword
- Packages & Access modifiers
- Inner classes

---

## 📁 03-collections-and-generics

Most frequently used data structures in real-world Java.

### Topics included

- ArrayList
- LinkedList
- Vector & Stack
- HashSet & TreeSet
- HashMap & TreeMap
- Queue & PriorityQueue
- Iterator & ListIterator
- Generics
- Custom generic classes
- Comparable & Comparator
- Sorting collections

---

## 📁 04-exception-handling-and-files

Write reliable, safe, production-ready Java code.

### Topics-included

- Exception types (checked & unchecked)
- `try`, `catch`, `finally`
- `throw` & `throws`
- Creating custom exceptions
- Java error hierarchy
- File handling
  - Reading files
  - Writing files
  - BufferedReader / FileReader
  - FileWriter / PrintWriter
- Serialization (optional)

---

## 📁 05-threads-and-advanced-java-basics

Modern Java features and multi-threading fundamentals.

### **Topics-included**

- Thread class
- Runnable interface
- Thread lifecycle
- Synchronization
- Locks
- ExecutorService (Thread Pool)
- Lambdas
- Functional interfaces
- Streams API (filter, map, reduce)
- Method references

---
